# 高情商聊天助手 - 用户手册

## 快速入门

### 首次使用

#### 1. 注册账号
1. 打开应用，进入注册页面
2. 填写用户名、密码和岗位信息
3. 点击"注册"按钮完成注册
4. 系统会自动为您设置默认的语气偏好和长度偏好

#### 2. 登录系统
1. 在登录页面输入用户名和密码
2. 点击"登录"按钮
3. 系统会记住您的登录状态，下次可自动登录

### 界面概览

#### 底部导航栏
- **首页**：查看推荐话术和系统公告
- **对话**：AI智能对话功能
- **话术**：浏览和搜索话术库
- **收藏**：查看个人收藏的话术
- **我的**：个人设置和账户管理

## 功能详解

### AI智能对话

#### 功能介绍
AI智能对话是本应用的核心功能，通过自然语言理解技术，根据您描述的沟通场景，自动生成适合的高情商话术。

#### 使用步骤

**步骤1：描述场景**
在对话界面输入您遇到的沟通场景，例如：
- "我需要拒绝同事的不合理请求"
- "我要向领导汇报项目风险"
- "客户对我们的价格有异议"

**步骤2：AI分析**
AI会分析您的场景描述，识别：
- 沟通对象（同事、领导、客户等）
- 沟通目的（拒绝、汇报、说明等）
- 情绪基调（委婉、专业、温和等）

**步骤3：生成话术**
AI会生成多个话术选项，每个选项包含：
- 话术内容：具体的沟通语句
- 适用场景：该话术最适合的使用场景
- 语气风格：话术的语气特点

**步骤4：选择调整**
- 如果有满意的话术，直接点击"使用"或"复制"
- 如果需要调整，点击"调整"按钮，选择不同的语气或长度

#### 高级功能

**个性化设置**
- 在"我的"页面设置您的岗位和语气偏好
- AI会根据您的设置优化话术生成

**对话历史**
- 系统会保存您的对话历史
- 可以回顾之前的对话和生成的话术

**话术收藏**
- 将满意的AI生成话术添加到收藏
- 方便后续快速查找和使用

#### 使用技巧

**场景描述建议**
1. **具体明确**：详细描述沟通场景和对象
   - ✅ 好的描述："我需要向产品经理反馈一个需求变更的问题，这个变更会影响开发进度"
   - ❌ 不好的描述："我需要反馈问题"

2. **包含关键信息**：
   - 沟通对象：同事、领导、客户
   - 沟通目的：拒绝、请求、说明、汇报
   - 具体情境：时间紧迫、关系紧张、需要维护关系

3. **表达期望**：
   - "我希望委婉地拒绝"
   - "需要保持专业态度"
   - "希望对方能够理解"

### 话术库浏览

#### 功能介绍
话术库包含了丰富的预设话术，按照岗位和场景进行分类，方便您快速查找和使用。

#### 浏览方式

**按岗位浏览**
1. 进入"话术"页面
2. 在顶部选择您的岗位
3. 系统会显示该岗位相关的所有话术

**按场景浏览**
1. 在"话术"页面选择场景分类
2. 常见场景包括：
   - 需求对接
   - 项目推进
   - 问题沟通
   - 协同配合
   - 会议沟通

**按语气浏览**
1. 在筛选条件中选择语气风格
2. 可选语气：温和、专业、委婉、活泼

#### 搜索功能

**关键词搜索**
1. 在搜索框输入关键词
2. 支持搜索话术标题和内容
3. 实时显示搜索结果

**组合搜索**
1. 结合岗位、场景、语气进行组合筛选
2. 精确定位您需要的话术

#### 话术详情

**查看详情**
1. 点击话术卡片查看完整内容
2. 详情页包含：
   - 完整话术内容
   - 适用场景说明
   - 使用建议和注意事项

**话术操作**
- **复制**：一键复制话术内容
- **收藏**：添加到个人收藏
- **分享**：分享给同事或朋友
- **调整**：基于当前话术进行个性化调整

## 使用场景示例

### 场景1：向同事求助

**场景描述**：
"我需要向同事请教一个技术问题，但不想打扰对方工作"

**AI生成话术**：
```
你好，关于[技术问题]，想请教一下你，方便吗？如果不方便的话，我可以等你有空的时候再问。
```

**使用建议**：
- 选择合适的时间：避免在对方忙碌时打扰
- 说明问题重要性：让对方了解问题的紧迫性
- 表达感谢：感谢对方的帮助

### 场景2：拒绝不合理请求

**场景描述**：
"同事让我帮忙做一个额外的工作，但我自己的任务还没完成"

**AI生成话术**：
```
我理解你的需求，不过目前手头有几个紧急任务要处理，可能暂时无法配合。等这周忙完，如果还需要的话可以再联系我。
```

**使用建议**：
- 理解对方立场：先表达理解
- 说明客观情况：说明自己的工作状况
- 提供替代方案：给出其他可能的解决方式
- 保持友好态度：维护良好的工作关系

### 场景3：向上级汇报问题

**场景描述**：
"项目出现了风险问题，需要向领导汇报"

**AI生成话术**：
```
领导，目前项目整体推进正常，但有一个风险点需要同步给您：[具体风险]。我们已经制定了应对方案，会密切跟进。
```

**使用建议**：
- 先说总体情况：让领导了解项目整体状况
- 重点突出风险：清晰说明风险的具体情况
- 提供解决方案：不只是提出问题，还要给出解决方案
- 表明跟进态度：让领导知道你在积极处理

### 场景4：处理客户异议

**场景描述**：
"客户对我们的产品价格有异议"

**AI生成话术**：
```
我非常理解您对价格的顾虑。其实我们的定价是综合考虑了产品质量、服务保障和长期价值。如果您有预算限制，我们也可以探讨一些灵活的合作方案。
```

**使用建议**：
- 表达理解：让客户知道你理解他的顾虑
- 解释价值：从客户角度说明产品价值
- 提供灵活性：给出多种合作选择
- 保持专业：始终保持专业态度

## 常见问题解答

### 注册与登录

**Q：忘记密码怎么办？**
A：目前系统暂不支持密码找回功能，建议您重新注册账号。后续版本会提供密码找回功能。

**Q：可以修改用户名吗？**
A：用户名一旦注册成功，暂不支持修改。如需修改，请联系客服处理。

**Q：手机号可以更换吗？**
A：可以在"我的"页面修改绑定手机号，需要验证原手机号。

### 功能使用

**Q：AI生成的话术不满意怎么办？**
A：您可以使用"调整"功能，选择不同的语气和长度，让AI重新生成。也可以手动修改话术内容。

**Q：如何找到适合自己岗位的话术？**
A：在"话术"页面选择您的岗位，系统会自动过滤出相关话术。也可以使用搜索功能搜索关键词。

**Q：收藏的话术会丢失吗？**
A：收藏的话术存储在云端，不会丢失。但建议定期导出重要话术作为备份。

**Q：可以分享话术给同事吗？**
A：可以，在话术详情页点击"分享"按钮，选择分享方式。支持微信、邮件等多种分享方式。

### 账户与设置

**Q：如何修改语气偏好？**
A：进入"我的"页面，点击"偏好设置"，选择您喜欢的语气风格。

**Q：可以同时收藏多个版本的话术吗？**
A：可以，您可以根据不同场景收藏不同版本的话术。

**Q：如何删除收藏的话术？**
A：在"收藏"页面，长按要删除的话术，选择"删除"选项。

### 技术问题

**Q：应用无法连接网络怎么办？**
A：请检查网络连接，确保手机有稳定的网络信号。如果问题持续，请尝试重启应用。

**Q：AI响应速度慢怎么办？**
A：AI响应速度受网络状况和服务器负载影响。建议在网络状况良好时使用。

**Q：如何清除缓存？**
A：在"我的"页面，点击"设置" > "清除缓存"，可以释放存储空间。
