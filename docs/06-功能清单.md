# 高情商聊天助手 - 功能清单

## 功能概述

本清单详细列出了高情商聊天助手的所有功能模块，包括已实现功能、开发中功能和计划功能。

## 功能分类

### 1. 用户认证模块

#### 1.1 用户注册
- **状态**：✅ 已实现
- **功能描述**：用户可以使用用户名、密码进行注册
- **详细功能**：
  - 用户名注册（3-50字符）
  - 密码设置
  - 手机号绑定（可选）
  - 邮箱绑定（可选）
  - 岗位选择
  - 语气偏好设置（默认：温和）
  - 长度偏好设置（默认：简洁版）
- **API接口**：POST /api/auth/register
- **前端页面**：Login.vue（注册Tab）

#### 1.2 用户登录
- **状态**：✅ 已实现
- **功能描述**：用户可以使用用户名、密码登录系统
- **详细功能**：
  - 用户名密码登录
  - JWT Token生成
  - Token自动刷新
  - 登录状态保持
  - 自动登录功能
- **API接口**：POST /api/auth/login
- **前端页面**：Login.vue（登录Tab）

#### 1.3 用户信息管理
- **状态**：✅ 已实现
- **功能描述**：用户可以查看和管理个人信息
- **详细功能**：
  - 查看用户基本信息
  - 修改手机号
  - 修改邮箱
  - 修改岗位
  - 上传头像
  - 修改语气偏好
  - 修改长度偏好
- **API接口**：
  - GET /api/auth/me
  - PUT /api/auth/me
- **前端页面**：Profile.vue

#### 1.4 用户登出
- **状态**：✅ 已实现
- **功能描述**：用户可以安全登出系统
- **详细功能**：
  - 清除本地Token
  - 清除用户状态
  - 跳转到登录页
- **前端页面**：各页面登出按钮

### 2. AI对话模块

#### 2.1 智能对话
- **状态**：✅ 已实现
- **功能描述**：基于用户输入的场景描述，AI自动生成适合的沟通话术
- **详细功能**：
  - 自然语言理解
  - 场景意图识别
  - 多话术选项生成
  - 上下文对话管理
  - 对话历史保存
- **API接口**：POST /api/chat/send
- **前端页面**：Chat.vue

#### 2.2 话术调整
- **状态**：✅ 已实现
- **功能描述**：根据用户需求调整话术的语气和长度
- **详细功能**：
  - 语气调整（温和、专业、委婉、活泼）
  - 长度调整（简洁版、标准版、详细版）
  - 实时预览调整结果
  - 调整历史记录
- **API接口**：POST /api/scripts/adjust
- **前端页面**：Chat.vue（话术卡片调整按钮）

#### 2.3 对话历史
- **状态**：✅ 已实现
- **功能描述**：查看和管理对话历史记录
- **详细功能**：
  - 对话历史列表
  - 会话管理
  - 历史搜索
  - 历史删除
- **API接口**：GET /api/chat/history
- **前端页面**：Chat.vue（历史记录）

#### 2.4 会话管理
- **状态**：✅ 已实现
- **功能描述**：管理多个对话会话
- **详细功能**：
  - 创建新会话
  - 切换会话
  - 会话重命名
  - 删除会话
- **API接口**：包含在对话接口中
- **前端页面**：Chat.vue（会话列表）
