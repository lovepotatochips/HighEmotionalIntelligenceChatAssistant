## 功能总览

本系统共包含7大功能模块，涵盖用户管理、AI对话、话术管理、收藏管理、系统配置等方面。

| 序号 | 功能模块 | 功能数量 | 优先级 |
|-----|---------|---------|--------|
| 1 | 用户管理模块 | 4 | 高 |
| 2 | AI智能对话模块 | 3 | 高 |
| 3 | 话术库管理模块 | 5 | 高 |
| 4 | 话术调整模块 | 3 | 中 |
| 5 | 话术收藏模块 | 3 | 中 |
| 6 | 对话历史模块 | 2 | 低 |

## 1. 用户管理模块

### 1.1 用户注册

| 功能点 | 说明 |
|-------|------|
| 功能编号 | F-U-001 |
| 功能描述 | 新用户注册账号 |
| 实现状态 | 已实现 |
| 功能详情 | 用户填写用户名、密码、手机号、邮箱、岗位角色、语气偏好、长度偏好等信息完成注册 |
| 输入参数 | 用户名（必填）、密码（必填）、手机号（可选）、邮箱（可选）、岗位角色（可选）、语气偏好（可选）、长度偏好（可选） |
| 输出结果 | 注册成功返回用户信息，失败返回错误提示 |
| 业务规则 | 用户名唯一、手机号唯一、密码加密存储 |
| 优先级 | 高 |

### 1.2 用户登录

| 功能点 | 说明 |
|-------|------|
| 功能编号 | F-U-002 |
| 功能描述 | 已注册用户登录系统 |
| 实现状态 | 已实现 |
| 功能详情 | 用户通过用户名和密码登录，成功后返回JWT令牌 |
| 输入参数 | 用户名、密码 |
| 输出结果 | 登录成功返回JWT令牌和用户信息，失败返回错误提示 |
| 业务规则 | 密码错误3次锁定账户30分钟、账户被禁用无法登录 |
| 优先级 | 高 |

### 1.3 获取用户信息

| 功能点 | 说明 |
|-------|------|
| 功能编号 | F-U-003 |
| 功能描述 | 获取当前登录用户信息 |
| 实现状态 | 已实现 |
| 功能详情 | 通过JWT令牌获取当前用户的详细信息 |
| 输入参数 | JWT令牌（通过请求头传递） |
| 输出结果 | 返回用户完整信息 |
| 业务规则 | 令牌验证失败返回401错误 |
| 优先级 | 高 |

### 1.4 更新用户信息

| 功能点 | 说明 |
|-------|------|
| 功能编号 | F-U-004 |
| 功能描述 | 更新当前用户信息 |
| 实现状态 | 已实现 |
| 功能详情 | 用户可以修改手机号、邮箱、岗位角色、头像、语气偏好、长度偏好等信息 |
| 输入参数 | 需要修改的字段及值 |
| 输出结果 | 返回更新后的用户信息 |
| 业务规则 | 用户名不能修改、密码需要单独接口修改 |
| 优先级 | 高 |

## 2. AI智能对话模块

### 2.1 发送消息获取AI回复

| 功能点 | 说明 |
|-------|------|
| 功能编号 | F-A-001 |
| 功能描述 | 用户发送消息，AI返回推荐话术 |
| 实现状态 | 已实现 |
| 功能详情 | 用户描述沟通场景，AI自动识别场景、意图、岗位，返回匹配的话术列表 |
| 输入参数 | 消息内容、会话ID（可选）、岗位（可选）、语气（可选）、长度（可选） |
| 输出结果 | AI回复内容、推荐话术列表、会话ID、识别的意图 |
| 业务规则 | 自动保存对话记录、支持多轮对话、上下文管理 |
| 优先级 | 高 |

### 2.2 话术场景识别

| 功能点 | 说明 |
|-------|------|
| 功能编号 | F-A-002 |
| 功能描述 | 自动识别用户描述的沟通场景 |
| 实现状态 | 已实现 |
| 功能详情 | 通过关键词匹配和上下文分析，识别沟通场景类型 |
| 支持场景 | 需求沟通、项目推进、问题沟通、协同配合、对外衔接、会议沟通 |
| 识别方式 | 关键词匹配、语义分析 |
| 优先级 | 高 |

### 2.3 岗位识别

| 功能点 | 说明 |
|-------|------|
| 功能编号 | F-A-003 |
| 功能描述 | 识别对话中提到的岗位 |
| 实现状态 | 已实现 |
| 功能详情 | 根据用户设置的岗位角色和对话内容，识别当前沟通岗位 |
| 支持岗位 | 售前人员、项目经理、产品经理、前端开发、后端开发、UI设计师、测试工程师 |
| 识别方式 | 用户设置优先、关键词匹配 |
| 优先级 | 高 |
